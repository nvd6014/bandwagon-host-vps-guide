## 先搞清楚：搬瓦工的套餐体系

搬瓦工的套餐按线路质量分四个层次，选之前要对自己的需求有个判断。

**KVM基础套餐**：走普通线路，洛杉矶、新泽西、温哥华等机房可选。适合预算紧张、对线路质量没特别要求的用户，比如学Linux、做备份、跑自动化脚本。

**CN2 GIA-E套餐**：主力系列。走CN2 GIA三网直连（电信CN2 GIA、联通9929、移动CMIN2），购买后可在12+个机房之间后台自由切换，包括DC6 CN2 GIA-E、DC9 CN2 GIA、日本大阪软银、荷兰联通等。性价比最高，是大多数国内用户的首选。

**香港/日本东京高端套餐**：延迟最低，香港机房到大陆可以做到个位数毫秒，但价格明显更高，适合对延迟非常敏感的场景，比如实时游戏、视频直播、企业内网接入。

**迪拜/悉尼等地区特供套餐**：面向特定地区用户，偶尔出现，按需了解。
